# Security Policy

## Supported Versions

Security fixes are supported for the current unreleased branch and releases that run on Python 3.10 or newer.

## Reporting a Vulnerability

Please do not open public issues for vulnerabilities.

Use GitHub private vulnerability reporting for this repository first. If private vulnerability reporting is unavailable, email `d.westermann@ol-mg.de` with the subject prefix `[SECURITY] YouTube_Batch_Downloader`.

Include:

- Affected version or commit.
- Reproduction steps or proof of concept.
- Expected impact.
- Any known workarounds.
